* Instrucciones de configuración *
**********************************
Para que la app funcione necesitamos instalar la BD en un mysql
Para ello, hay que modificar el usuario y contraseña del archivo config.php
Actualmente se entrega con las credenciales de ejemplo localhost del xampp
Luego es solo correr la app ya sea en un ambiente local o en una url publica.
Actualmente se puede acceder a esta version en

https://marcexlweb.com/felipe 

*************
* Contenido *
*************
La aplicacion es una primera version prototipo de juego simple
Donde por le momento de forma aleatoria muestra diferentes lugares de 
Donde esta Felipe.
El objetivo es poder sumar puntos o restar dependiendo lo que necesite el gato.
Actualmente echo en PHP, HTML, CSS, JS con Jquery
